2024-09-01

Preserving Hakka

This idea come from a endangered phenomenon happening in my hometown, my younger sisters and brothers used to speak Hakka,
but after they come to live with their parents in Shenzhen, they can’t speak it anymore, and even can’t understand some
sentences.
Also, newly borned babies in my hometown can only speak Mandarin because their kindergarten teachers and classmates are all
speaking Mandarin. This for me is a extremely bad signal, maybe one day Hakka would also go extinct.
So I decided to use TTS model to preserve Hakka, and it’s also a solution to save endangered languages all around the world.

2024-07-01

Assistant for Reading arXiv Papers

To have a deeper understanding of how state-of-art models are disigned, and what tricks can be used
to train these models, we always have to look at the papers written by the authors.
During this period, I always find some difficulty in truly understanding the key points behind the
truly long context in the paper, so I turned to ask help for LLM.
Fortunately, Llama3.1 was published at this moment, so I create a space in huggingface

In this space, the model can help me make a summary of the chosen arXiv paper and answer my questions
based on each chapter.
and also if you have any questions after reading this chapter, or didn’t what to ask, that’s fine, just click…
A beautiful UI can also help me focus.

2023-11-01

Implementing basic machine learning algorithms in numpy

When I was studying CS229 by Andrew Ng, I found that by coding when studied theory can make a better understanding,
so I make a github repository

In this repository, most famous machine learning algorithms like svm, svr, random forest, xgboost can be found,
and other unsupervised learning algorithms like ICA, Gaussian Mixture Model can be found as well.
For deep learning, I also implemented a simple neural network and a simple CNN, by writing backpropagation in the draft first and code them.
And the most important thing, they are all implemented by numpy.
It is a interesting journey to implement these algorithms…
