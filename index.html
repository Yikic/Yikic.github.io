Preserving Hakka

This idea comes from an endangered phenomenon happening in my hometown. My younger siblings used to speak Hakka, but after moving to Shenzhen to live with their parents, they have almost lost the ability to speak it, and even can’t understand some sentences.

Also, the new generations in my hometown only speak Mandarin, because their teachers and classmates at kindergarten all use Mandarin. This, to me, is an alarming signal—which suggests that Hakka could face extinction someday.

To help preserve Hakka, I decided to use a TTS (Text-to-Speech) model. I believe this approach could also be a way to save other endangered languages all around the world.

2024-07-01

Assistant for Reading arXiv Papers

Understanding state-of-the-art models and the techniques used to train them often requires diving into papers written by the authors themselves. However, extracting the key points within lengthy and complex papers can be quite challenging. That’s when I turned to LLM for help.

Recently, with the release of Llama 3.1, I decided to create a space on Huggingfaceto help me better understand arXiv papers.

This space leverages the power of LLM to summarize selected arXiv papers and answer questions based on each chapter, making it easier to grasp the main ideas.

If you have any questions after reading a chapter, or even if you’re not sure what to ask, that’s oookay—just click and explore!

A clean and intuitive UI also helps me stay focused.

2023-11-01

Implementing basic machine learning algorithms in NumPy

While studying CS229 by Andrew Ng, I found that coding alongside learning theory greatly enhances understanding. This led me to create a GitHub repository where I implemented various machine learning algorithms using NumPy.

In this repository, implementations of some of the most well-known machine learning algorithms can be found, such as SVM, SVR, Random Forest, and XGBoost, as well as several unsupervised learning algorithms like ICA and Gaussian Mixture Models.

For deep learning, I have also included a basic neural network and a simple CNN, both implemented by first writing out the backpropagation outcomes on the draft and then coding them in NumPy.

It is a interesting journey to implement these algorithms…
